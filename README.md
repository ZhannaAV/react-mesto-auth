# Проект 12: Место

*Проектная работа Mesto на React*

## Команды

`npm start` — запускает приложение на локальном сервере.

`npm run build` — сборка продакшен билда.

### **Прорабатываемые технологии**

* Регистрация/авторизация
* Кэширование
* Хуки
* Глобальный контекст
* API
* Бургер-меню
* Адаптивная вёрстка
